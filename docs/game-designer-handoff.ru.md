# Документ для гейм-дизайнера

Извлечено из текущего кода игры без изменений геймплейного кода.

Основные исходные файлы:
- `CardDataGenerator.js`: веса карт по этажам, враги, боссы, снаряжение, ловушки, пул амулетов, зелья, еда, магические карты.
- `cardSystem.js`: размер поля, правила плотности врагов, дроп при смерти врагов, награда за мимика, генерация карт этажа.
- `gameScene.js`: награды за зачистку этажа и комнаты наград после боссов.
- `scenes/ShopScene.js`, `scenes/RareShopScene.js`, `scenes/TreasureScene.js`, `scenes/RestScene.js`, `scenes/AnvilScene.js`: экономические комнаты.
- `scenes/EventScene.js`: выборы и награды в событиях.
- `AmuletManager.js`: эффекты амулетов.
- `MetaProgressionManager.js`: реликвии за смерть.

## Общая структура

| Система | Текущее поведение |
|---|---|
| Длина забега | 45 этажей, 3 акта по 15 этажей |
| Этажи боссов | 15, 30, 45 |
| Старт игрока | 115 HP, 15 AP, 0 монет, 0 кристаллов, 5 слотов инвентаря |
| Прогрессия | Лут по этажам, слияние снаряжения, амулеты, реликвии, исходы событий |

| Основная награда за бой | Выдается один раз, когда все враги побеждены |

## Узлы карты

Карта каждого акта имеет 15 этажей. Этаж 0 - фиксированный стартовый боевой узел; этаж 14 - узел босса. Средние этажи генерируют по 2-4 узла.

| Тип узла | Примечания |
|---|---|
| COMBAT | Обычный бой |
| ELITE | Более сложный бой, после зачистки открывает элитный сундук |
| SHOP | Обычный магазин за монеты/кристаллы |
| RARE_SHOP | Премиум-магазин; минимум один на акт |
| REST | +20 HP, AP полностью восстанавливаются |
| ANVIL | Станция ремонта; гарантированно две на акт |
| EVENT | Случайное событие |
| TREASURE | Сундук с ключом или силовым открытием |
| TREASURE_GOOD | Улучшенный сундук, более редкий |
| BOSS | Босс акта |

Веса комнат карты до ограничений размещения:

| Тип | Вес |
|---|---:|
| COMBAT | 48 |
| ELITE | 8 |
| SHOP | 10 |
| RARE_SHOP | 4 |
| REST | 12 |
| ANVIL | 8 |
| EVENT | 21 |
| TREASURE | 4 |
| TREASURE_GOOD | 2 |

Гарантии на акт: минимум один отдых, магазин, редкий магазин, обычный сундук, хороший сундук, одна ранняя наковальня и одна поздняя наковальня.

## Веса выпадения типов карт

Это веса, а не финальные точные проценты. Итоговые поля могут отличаться из-за лимитов, минимумов врагов, максимумов врагов и гарантий оружия.

Для этажей 1-30 есть предзаданные веса; этаж 31+ использует формулу ниже. Этажи боссов: `{ boss: 100 }`.

После балансировки вес каждого не-боссового этажа корректируется:

| Тип | Корректировка |
|---|---|
| enemy | Этаж <= 14: x0.68, этаж >= 15: x0.78, минимум 20 |
| coin | x0.25, минимум 1 |
| trap | x0.75, минимум 3 |
| weapon | x0.95 плюс бонус этажа; минимум 7/9/11/12 по диапазонам этажей |
| armor | x1.15, минимум 10 до этажа 18, 12 после |
| amulet | x0.4, минимум 1, максимум 4 до этажа 15 и 6 после |
| potion | x1.2, минимум 8 |
| food | x1.7, минимум 19 |
| magic | x1.25, минимум 5 |
| thorns | минимум 3 |
| crystal | x0.8 до этажа 15, x0.5 после, минимум 3 |
| gem | 0 до этажа 12, 3 на этажах 12-15, 9 на этажах 16-30, 12 на этажах 31+ |
| key | минимум 2 |
| empty | 0 до этажа 15 включительно, затем минимум 12 |

Формульные веса для этажей, не указанных явно:

| Тип | Формула |
|---|---|
| enemy | `min(30 + floor * 2, 82)` |
| coin | `max(30 - floor, 2)` |
| crystal | `min(5 + floor * 0.4 с округлением вниз, 18)` |
| trap | `min(5 + floor, 35)` |
| weapon | `max(10 - floor / 4 с округлением вниз, 1)` |
| armor | `min(6 + floor / 3 с округлением вниз, 18)` |
| amulet | `min(4 + floor * 0.8 с округлением вниз, 35)` |
| potion | 10 |
| food | `max(12, 18 - floor / 3 с округлением вниз)` |
| magic | `min(3 + floor / 2 с округлением вниз, 15)` |
| gem | 9 до балансировки, затем перезаписывается гейтом самоцветов |
| key | `min(1 + floor / 4 с округлением вниз, 2)` |
| mimic | 3 с этажа 3+ |

## Реликвии за смерть

Реликвии открываются после смерти. Для обычных врагов совпадение широкое: любое имя убийцы, содержащее spider, skeleton или goblin, может выдать следующую неоткрытую реликвию этой семьи врагов.

| Смерть или веха | Реликвия | Эффект |
|---|---|---|
| Смерть от spider | Spider Venom | Оружие имеет 20% шанс отравить врагов: 2 урона на 3 хода |
| Следующая смерть от spider | Webweaver's Thread | 10% шанс, что слитая карта возродится рубашкой вверх на поле |
| Смерть от skeleton | Bone Armor | Начинать каждый забег с Common Bone Armor, 2 защиты, 15 прочности |
| Следующая смерть от skeleton | Gravebloom Bundle | Лечит 2 HP в начале каждого боевого этажа |
| Смерть от goblin | Goblin War Horn | Первая атака на каждом этаже наносит двойной урон |
| Следующая смерть от goblin | Scavenger's Coffer | +20% монет из всех источников |
| Смерть от Giant Skeleton | Giant's Morningstar | Все оружие наносит +1 урон |
| Смерть от Spider Queen | Queen's Antivenom | Иммунитет к яду |
| Смерть от Lich | Lich's Covenant | Лечит 1 HP за убитого врага, -10 макс. HP |
| 3 смерти всего | Ironhide Tonic | Броня теряет прочность вдвое реже |
| 5 смертей всего | Veteran's Carryall | +1 постоянный слот инвентаря |
| Лучший этаж 7+ | Wayfarer's Camp | +1 макс. HP, когда карта с прочностью полностью израсходована |
| Лучший этаж 10+ | Dungeonmaster's Spectacles | Видеть одну дополнительную карту в начале |
| После получения всех реликвий | Резервный Veteran HP | +2 постоянных макс. HP за подходящую смерть, максимум +20 |

## Combat / оружие

  на карте совпадала с ударом. Нужно отдельно продумать weakened (и любые

- **[ ] Вес блокирования луком.** Сейчас блок тратит ход (AP + durability) и

  «раунд» после этого идёт как обычно. Неясно, достаточно ли это сильно /
  слабо, и не нужно ли менять модель (блок одного удара, частичный блок,
  стоимость, длительность).

## Броня

- **[ ] Доработать chain / plate перед возвратом в пул дропа.** Пока в игре
  спавнится только Leather (dodge-only, без protection; durability −1 на
  успешный dodge). Набросок направлений:
  - **Chain (кольчуга):** без dodge; одна прочность тратится за N атак
    врагов в зависимости от редкости — **2 / 3 / 4 / 5 / 6**
    (common → legendary).
  - **Plate (сталь):** без dodge; одна прочность блокирует **1 / 2 / 3 / 4 / 5**
    урона от *одного* врага (common → legendary).
  - Уточнить взаимодействие с armor_break, companion protection, dodge-амулетами
    и отображением в UI.

## Гемы

- **[ ] Гемы, которые не влезают в новое оружие при слиянии.** Слоты =
  1 / 2 / 3 / 4 / 5 по редкости. При merge сумма стаков / разные типы могут
  превысить слоты результата. Скорее всего нужен экран выбора «какие оставить»
  (и что происходит с отброшенными — возврат в инвентарь / потеря / монеты).

- **[ ] Как усиливать гемы при слиянии одинаковых?** Пока стаки просто
  складываются (с clamp по слотам новой редкости), а кривая силы эффекта
  для стаков 4–5 — временная линейная. Нужно решить: отдельный tier-ап гем-эффекта,
  бонус к урону, новые проки, или только больше слотов без роста силы за стак.
